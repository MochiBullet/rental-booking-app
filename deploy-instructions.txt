車・バイクレンタル予約サイト「RentalEasy」公開手順

=== 最も簡単な方法 ===
1. https://app.netlify.com/drop を開く
2. rental-site.zip をドラッグ&ドロップ
3. 数秒で公開完了！

=== 準備済みファイル ===
- rental-site.zip : 公開用圧縮ファイル
- build フォルダ : 公開用ファイル一式

=== GitHub Pages での自動公開 ===
GitHub アカウントがある場合：
1. GitHubでリポジトリ作成
2. コードをプッシュ
3. npm run deploy で自動公開
