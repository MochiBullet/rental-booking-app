AWS自動セットアップツール

⚠️ 前提条件: AWSアカウントが必要です。まだお持ちでない場合は こちらから無料で作成できます。

1リージョンを選択

デプロイ先のAWSリージョンを選択してください：

2IAMユーザーを自動作成

下のボタンをクリックして、AWSコンソールでCloudFormationスタックを作成します。

これにより、CDKデプロイに必要なIAMユーザーとアクセスキーが自動的に作成されます。

ボタンをクリック後：
1. AWSコンソールにログイン
2. CloudFormationの画面が開きます
3. 「スタックの作成」をクリック
4. 完了まで約2分待ちます

3認証情報を保存

CloudFormationスタックが完了したら、「出力」タブから以下の値をコピーして保存してください：

重要: Secret Access Keyは一度しか表示されません。必ず安全な場所に保存してください。

4環境変数の設定

Windows PowerShellまたはコマンドプロンプトで以下のコマンドを実行：

# PowerShellの場合 $env:AWS_ACCESS_KEY_ID="YOUR_ACCESS_KEY_ID" $env:AWS_SECRET_ACCESS_KEY="YOUR_SECRET_ACCESS_KEY" $env:AWS_DEFAULT_REGION="ap-northeast-1" # または永続的に設定 [System.Environment]::SetEnvironmentVariable("AWS_ACCESS_KEY_ID", "YOUR_ACCESS_KEY_ID", "User") [System.Environment]::SetEnvironmentVariable("AWS_SECRET_ACCESS_KEY", "YOUR_SECRET_ACCESS_KEY", "User") [System.Environment]::SetEnvironmentVariable("AWS_DEFAULT_REGION", "ap-northeast-1", "User")

5CDKをデプロイ

認証情報設定後、以下のコマンドでアプリケーションをデプロイ：

cd rental-booking-app\cdk deploy.bat
✅ 完了! CDKデプロイが成功すると、アプリケーションのURLが表示されます。 GitHub Actionsも自動的に設定され、以降はコードをプッシュするだけで自動デプロイされます。
